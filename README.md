# **研究所入學趨勢分析**

此專題為個人專題，蒐集了近五年部分大學的資工所和資管所的甄試及考	試入學的報名人數和錄取人數，使用R語言繪製成可視覺化的圖表，觀察	並探討其中的趨勢。
